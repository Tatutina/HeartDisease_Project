# HeartDisease_Project
Разработка системы прогнозирования риска сердечно-сосудистых заболеваний на основе предобработки данных, статистического анализа и моделей машинного обучения для выявления ключевых факторов риска у пациентов с нарушениями углеводного обмена.

# Heart Disease Risk Analysis Project

Проект разделён на три основных части:

1. **Предобработка данных** - очистка, заполнение пропусков, кодирование категориальных признаков.
2. **Статистический анализ** - описание данных, визуализации, проверки гипотез.
3. **Машинное обучение** - построение моделей классификации, оценка качества.

## Запуск проекта

1. Клонируйте репозиторий
2. Установите зависимости:
   preprocessing/ — скрипты и ноутбуки для предобработки данных: очистка, заполнение пропусков, кодирование признаков.
   statistical_analysis/ — статистический анализ данных: описательные статистики, графики, тестирование гипотез.
   machine_learning/ — модели машинного обучения: обучение, настройка, оценка качества.
   data/ — исходные и подготовленные данные.
   reports/ — итоговые отчёты и визуализации в формате PDF или HTML.
   requirements.txt — список используемых библиотек.
   README.md — этот файл с описанием проекта и инструкциями.
3. Последовательно запустите скрипты из папок `preprocessing/`, `statistical_analysis/`, `machine_learning/`.
4. Ознакомьтесь с отчетами в папке `reports/`.
